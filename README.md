# Ejercicio Hilos, Multiprocesos y Secuenciales
Proyecto: Laboratorio de Arquitectura de Computadores “Monitor de recursos del sistema con concurrencia y gráficos en tiempo real”

Este aplicativo tiene un dashboard para visualizar CPU, RAM, núcleos y sistema en tiempo real. 
Incluye procesamiento de imágenes y un log de ejecución donde verás cuánto tiempo tardó cada modelo de concurrencia en procesar cada imagen
<br> <br> El log también muestra información sobre la imagen más rápida y el estado de la RAM y CPU antes y después de la ejecución. 
Al finalizar los tres modelos, aparecerá una tabla comparativa de consumo. Se sugiere correrlo en Spyder 6 para no instalar nada extra, 
te recomiiendo crear una carpeta para cada modelo
